Maven
dependency
puzzlers

by Geoffrey De Smet

Every question is the same:

Which jars end up
in the production classpath
of myproject?

Which jars on the classpath?

After running

$ mvn assembly:single

on myproject
with assembly configuration 

<assembly>

  ...

  <dependencySets>
    <dependencySet>
      <outputDirectory/>
    </dependencySet>
  </dependencySets>

</assembly>

Environment

  • Maven 3.0.4
  • OpenJDK 1.7
  • Linux

Different groupId's

<dependencies>
  <dependency>
    <groupId>javassist</groupId>
    <artifactId>javassist</artifactId>
    <version>3.12.1.GA</version>
  </dependency>
  <dependency>
    <groupId>org.javassist</groupId>
    <artifactId>javassist</artifactId>
    <version>3.16.1-GA</version>
  </dependency>
</dependencies>
  • A) javassist-3.12.1.GA.jar
  • B) javassist-3.16.1-GA.jar
  • C) javassist-3.12.1.GA.jar
     + javassist-3.16.1-GA.jar
  • D) neither
  • E) I don't know
